# GLaDOS Emporium Bash Application

> A bash application that is a store themed after the A.I. character from the video game series Portal. It has a account store system that has the basic login, signup, and logout functionality.

---

## Application Structure

1. Assistant (Adam) welcomes user to store and then prompts if the user wants to go to the store or learn about the store.
   - Learning about the store will explain how the store is structured and how to purchase items with a dummy account.
   - Going straight to the store will then prompt the user if they have an account
2. Assistant then asks user if they have an account or not.
